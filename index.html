Welcome to Jon's raspbian repository

This site hosts get-iplayer packages for Raspbian on the Raspberry Pi, including backports of tools it requires:

  • get-iplayer
  • AtomicParsley

Recent Changes

Sep 2017: Jessie Packages Retired

Jessie is no longer supported - please re0run the install script below. (It's not sufficient to update your apt sources to point to stable, the repository signing key has changed too.)

ffmpeg and rtmpdump are no longer packaged here for raspbian since Stretch includes a recent version of ffmpeg, and rtmpdump is no longer used by get-iplayer. AtomicParsley has now almost caught up in the debian respositories, so no further versions will be published here.

Using this repository

To install the repository, get-iplayer and its dependencies, paste this line into a terminal window:

wget http://packages.hedgerows.org.uk/raspbian/install.sh -O - | sh

To remove get-iplayer and the repository, paste this line into a terminal window:

wget http://packages.hedgerows.org.uk/raspbian/uninstall.sh -O - | sh

Both scripts use sudo for some tasks, which will ask for your password. If you want to know in advance what the script will do, see install.sh or uninstall.sh.
