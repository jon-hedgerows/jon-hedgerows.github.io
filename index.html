Welcome to Jon's raspbian repository

This site hosts get-iplayer packages for the Raspberry Pi (specifically, for Raspbian jessie), including backports of tools it requires:

  • get-iplayer
  • AtomicParsley
  • ffmpeg
  • rtmpdump

Recent Changes

Feb 2017: Wheezy Packages Retired

Wheezy is no longer supported - please update your apt sources to point to jessie instead - see step 1 below.

Jan 2017: Expired Repository Signing Key

My GPG key expired in January 2017 - if you installed the repository signing key prior to this you will need to reinstall it:

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv 0F5BFDFE

Using this repository

  1. Add the repository:

    Paste these lines into a terminal window -

    echo "deb http://packages.hedgerows.org.uk/raspbian jessie/" | sudo tee /etc/apt/sources.list.d/packages.hedgerows.org.uk.list
echo "deb-src http://packages.hedgerows.org.uk/raspbian jessie/" | sudo tee -a /etc/apt/sources.list.d/packages.hedgerows.org.uk.list

  2. Install the repository signing key: 
    sudo apt-key adv --keyserver keyserver.ubuntu.com --recv 0F5BFDFE

  3. Update your package index: 
    sudo apt update
  4. Install get-iplayer: 
    sudo apt install get-iplayer

Removing this repository

  1. Remove get-iplayer: 
    sudo apt purge get-iplayer
  2. Remove the repository: 
    sudo rm /etc/apt/sources.list.d/packages.hedgerows.org.uk.list

  3. Remove the repository signing key: 
    sudo apt-key del 0F5BFDFE

  4. Update your package index: 
    sudo apt update
